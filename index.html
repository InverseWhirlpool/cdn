Octofile is a full platform file transfer application that supports multiple file formats while maintaining a stable transfer speed.

# Supported features

  • Support for multiple platforms
  • Support for multiple types of documents
  • No limit on file size
  • Unrestricted transmission speed
  • Common equipment
  • Multilingual support
  • Support for self-built hotspots to transfer files
  • Support for sending files to Kindle

# Supported platforms

  • Android
  • iOS
  • macOS
  • Windows
