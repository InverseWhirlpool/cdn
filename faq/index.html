# What platforms are supported?

Currently supported are: Android, iOS, macOS, Windows

# Can't find the device when receiving files?

Please troubleshoot the problem in the following order:

  1. Check the wireless network connection to make sure the device is on the same WiFi network
  2. Receive by scanning a QR code
  3. If your router has a high load, it may not be able to find the device, please try to reboot the router
  4. Close the application and try again after the above steps.

# Unable to receive files?

Please troubleshoot the problem in the following order:

  1. Check the wireless network connection to make sure the device is on the same WiFi network
  2. Receive by scanning a QR code
  3. If your router has a high load, it may not be able to find the device, please try to reboot the router
  4. Close the application and try again after the above steps.

# Where are the received files stored?

Android: Located at /Android/data/com.hypobenthos.octofile/Downloads, you can use any file browser to access

iOS: Located in the sandbox of the application, you can use the Files app to access the folder with the app name under My iPhone

macOS: Located in the Downloads folder

Windows: Located in the sandbox of the application, you can access the active "Downloaded" button to open the folder and view the downloaded files

# What is designated device sending?

Specifying a device to send a file is an additional feature that simplifies the sending and receiving process through a message push service, increasing efficiency.

# What platforms are supported by the designated device?

Currently supported are: iOS, Android

We will provide support for other platforms as well in subsequent updates

# Why can't I get push messages from the sender?

iOS Uses APNs to register and send messages, please check if you have permission to enable notifications

Android Uses `FCM' to register and send messages, please check if you have permission to enable notifications, and add the app to the whitelist launched in the background

The client must be version 2.0 or above, and the device is in the same network environment

In some countries and areas, the inability to connect to APNs or FCM servers directly results in this feature not being available

# What is a downloadable list?

The downloadable list is an add-on feature that sends files to devices that support browsers. Such as mobile phones, tablets, computers, Kindle, etc.

If you don't have a client for Octofile on the receiving end, you can send files via this feature
