# 隐私政策

本应用非常重视您的隐私, 使用过程中所产生的文档数据只会存储在设备本地

注意

我们可能随时对隐私政策进行更新, 如果隐私政策作出重大变更, 我们将在 App 内通告更新的隐私政策

# 收集的个人信息

  • 安装UUID
  • IP 地址
  • 应用崩溃信息
  • 广告标识符

# 访问的个人信息

  • 文件读取和写入权限, 用于文件传输
  • 相册的读取和写入权限, 用于文件传输
  • 相机的访问, 用于扫描二维码
