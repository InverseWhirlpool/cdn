# 支持哪些平台?

目前受支持的有: Android、iOS、 macOS、Windows

# 接收文件时找不到设备?

请按以下顺序来排查问题:

  1. 检查无线网络连接, 确保设备是在同一个WiFi网络中
  2. 使用扫描二维码的方式进行接收
  3. 如果您的路由器负载较高, 可能会导致设备查找功能失效, 请尝试重启路由器
  4. 上述步骤操作完后请关闭应用后重试

# 无法接收文件?

请按以下顺序来排查问题:

  1. 检查无线网络连接, 确保设备是在同一个WiFi网络中
  2. 使用扫描二维码的方式进行接收
  3. 如果您的路由器负载较高, 可能会导致设备查找功能失效, 请尝试重启路由器
  4. 上述步骤操作完后请关闭应用后重试

# 已接收的文件存放在哪?

Android: 位于 /Android/data/com.hypobenthos.octofile/Downloads 路径下, 您可以使用任意文件浏览器访问

iOS: 位于应用的沙盒中, 您可以使用文件 应用访问我的iPhone 下对应应用名称的文件夹中

macOS: 位于下载文件夹中

Windows: 位于应用的沙盒中, 您可以访问活动中的已下载按钮来打开文件夹并查看已下载的文件

# 指定设备发送是什么?

指定设备发送文件是一个附加功能, 通过消息推送服务简化发送和接收的步骤, 提高效率

# 指定设备发送支持哪些平台?

目前受支持的有: iOS, Android

我们会在后续更新中为其他平台也提供支持

# 为什么我收不到发送端的推送消息?

iOS 使用了APNs 来注册和发送消息, 请检查是否开启通知的权限

Android 使用了FCM 来注册和发送消息, 请检查是否开启通知的权限, 并把应用加入后台启动的白名单

客户端必须是2.0及以上的版本, 且设备在同一个网络环境下

部分国家和地区因为无法连接APNs 或FCM 服务器会直接导致此功能无法使用

# 什么是可下载列表?

可下载列表是发送文件是一个附加功能, 通过这个功能可以发送到支持浏览器的设备. 如手机, 平板, 电脑, Kindle等

如果你的接收端没有章鱼快传的客户端, 您可以通过这个功能发送文件
