# Privacy policy

This application takes your privacy very seriously, the document data generated during use will only be stored locally on the device

WARNING

We may update our privacy policy from time to time, If there are significant changes to the privacy policy, We'll post an updated privacy policy within the App

# Personal information collected

  • Installation of a UUID
  • IP address
  • Crash information
  • Advertising identifiers

# Personal information accessed

  • File read and write permissions, for file transfer
  • Read and write permissions for albums, for file transfer
  • Camera access, for scanning QR codes
